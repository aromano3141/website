# Anthony Romano - Personal Portfolio

A modern, minimalist personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## Project Structure

-   **Frontend:** React (Vite)
-   **Styling:** Tailwind CSS
-   **Animations:** Framer Motion
-   **Icons:** Lucide React

## Getting Started

To run this project locally, follow these steps:

1.  **Install Dependencies:**

    ```bash
    npm install
    ```

2.  **Start the Development Server:**

    ```bash
    npm run dev
    ```

3.  **Open in Browser:**
    Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`).

## Features

-   **Hero Section:** Terminal-style typing effect for bio.
-   **About:** Concise professional summary.
-   **Experience & Publications:** Vertical timeline layout.
-   **Projects:** Card layout with tech stack badges.
-   **Skills:** Categorized grid layout.
-   **Dark Mode:** "Academic High-Tech" theme.
